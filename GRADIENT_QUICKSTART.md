# FlowAngle Gradient Shading - Quick Start Guide

## What This Is

A demo showcasing gradient-based shading for FlowAngle shapes with 4 different gradient types and 8 color schemes.

## Quick Start

1. **Open the file:**
   ```
   open /Users/preston/research-developer/svGen-shading/shading_test_gradients.html
   ```

2. **Try these combinations:**

   **Triquetra with Ocean Radial:**
   - Sides: 3
   - Curve: -0.66
   - Gradient Style: Radial
   - Color Scheme: Ocean Blue

   **Flower with Iridescent Segments:**
   - Sides: 6
   - Curve: -0.8
   - Gradient Style: Per-Segment
   - Color Scheme: Iridescent

   **Star with Fire:**
   - Sides: 5
   - Curve: -1.2
   - Gradient Style: Curve-Reactive
   - Color Scheme: Fire

## Gradient Styles

| Style | Description | Best For |
|-------|-------------|----------|
| **Radial** | From center outward | 3D depth, symmetric shapes |
| **Linear** | Follows rotation angle | Directional lighting |
| **Per-Segment** | Each curve segment | Prismatic, iridescent effects |
| **Curve-Reactive** | Adapts to curve factor | Dynamic morphing |

## Color Schemes

- **Ocean** - Deep blue to white
- **Sunset** - Warm orange/yellow
- **Forest** - Deep to light green
- **Metallic** - Industrial gray
- **Iridescent** - Rainbow (special with per-segment)
- **Fire** - Red to yellow flames
- **Ice** - Cool blues
- **Purple** - Violet dream

## Controls

- **Gradient Style** - Select gradient type
- **Color Scheme** - Choose color palette
- **Gradient Intensity** - Adjust opacity (0-1)
- **Sides, Curve, Angle, Rotation** - Shape parameters
- **Download SVG/PNG** - Export your creation

## Tips

- Try **Iridescent + Per-Segment** for rainbow effects
- Use **Curve-Reactive** and adjust curve slider to see it respond
- Lower intensity (0.3-0.5) for subtle effects
- Higher intensity (0.8-1.0) for bold colors
- Rotate while using Linear gradient to see it follow

## Advanced

The curve-reactive gradient is unique - it changes its angle based on how curved the shape is. More negative curve factor = more angle shift. Try it with the star preset!
