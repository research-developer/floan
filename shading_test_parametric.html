How to Read

Color Temperature: Indicates curve direction. Cool blues = negative curve, warm oranges = positive curve.

Brightness: Shows handle angle sharpness. Darker = sharper angles, brighter = gentler curves.

Saturation: Reflects shape complexity. More sides = more saturated colors.

Per-Segment: Each segment can show position-based variation in a circular pattern.
